---

## 📖 Про проект

Віртуальна примірочна з використанням **MindAR Face Tracking** та **A-Frame**.

Користувачі можуть:
- Приміряти різні аксесуари (капелюхи, окуляри, сережки)
- Перемикатися між аксесуарами через панель вибору
- Захоплювати фото з результатом примірки
- Ділитися фото через Web Share API

---

## 🎯 Функціонал

| Функція | Опис |
|---------|------|
| **Відстеження обличчя** | 468 опорних точок через MediaPipe |
| **Оклюзія** | Невидима модель голови для реалістичності |
| **5 аксесуарів** | 2 капелюхи, 2 окуляри, 1 сережки |
| **Панель вибору** | Категорії + мініатюри |
| **Захоплення фото** | Кнопка або клавіша C |
| **Web Share API** | Поширення фото через системні додатки |
| **Адаптивність** | Працює на мобільних та десктопах |

| Технологія | Версія | Призначення |

| **Web Share API** | — | Поширення файлів |
| **Canvas API** | — | Захоплення кадрів |

---

## 🚀 Запуск

1. Відкрийте `index.html` на GitHub Pages
2. Натисніть **"Віртуальна примірочна"**
3. Дозвольте доступ до камери
4. Наведіть камеру на обличчя
5. Вибирайте аксесуари з панелі вгорі
6. Натисніть кнопку захоплення або клавішу **C**
7. Поділіться фото через Web Share API
